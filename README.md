# book_sales
This repository is used for data analysis of Amazon book sales. The dataset was obtained from the @Practice-Datasets-for-Excel repository, and this project is intended for practice and learning purposes. 
[Dataset_Details.docx](https://github.com/user-attachments/files/24529874/Dataset_Details.docx)

I am starting to practice data analysis. I am connecting Power BI with Google Sheets to automate the process for free, as this is the only option available to me at the moment since I cannot use SQL or Python yet. I am using this project to learn data analysis from scratch.
